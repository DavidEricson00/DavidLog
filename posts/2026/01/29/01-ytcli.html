Como eu derrubei o yt-dlp ( história 100% real)

Esses dias eu tentei baixar um vídeo no YouTube pelo navegador como sempre fazia, mas, como normalmente acontece, deu erro ao baixar. Depois de tanto tempo usando esses downloaders de navegador, passando por anúncio, contagem regressiva, ou vendo sites eu já era habituado simplesmente caírem, decidi que ia tomar coragem e procurar uma alternativa local e gratuita pra isso. Foi aí que eu descobri o yt-dlp.

O yt-dlp é basicamente um downloader de vídeos de inúmeras páginas diferentes que pode ser executado facilmente pelo terminal. Ao começar a usar, achei bem eficiente, mas com alguns probleminhas que não são culpa da plataforma em si, mas sim da minha preguiça. Coisas como ter que ficar reescrevendo o comando completo pra baixar, ou não ter um diretório de download padrão. Diante dessas dificuldades, decidi criar um CLI simples em Python pra usar o downloader por baixo dos panos e atender todas as minhas demandas.

Rapidamente terminei o projeto, fiz inúmeros testes e todos funcionando perfeitamente. Todas as features implementadas, README feito, tudo funcionando sem problemas, publiquei o repositório no GitHub. Só faltava uma coisa: escrever um post pro LinkedIn. Afinal, tenho que mostrar que tô fazendo alguma coisa :D

Fiz o post completo e aí fui tirar algumas prints da aplicação rodando no terminal.

Mas aí…

Veio a mensagem:

ERROR: unable to download video data: HTTP Error 403: Forbidden

Pensei: como assim??? O projeto tava funcionando corretamente até agora… por que do nada deu esse erro? O que tá acontecendo???

Testei novamente:

ERROR: unable to download video data: HTTP Error 403: Forbidden

Voltei o código pra versões anteriores:

ERROR: unable to download video data: HTTP Error 403: Forbidden

Esperei um dia pra testar novamente:

ERROR: unable to download video data: HTTP Error 403: Forbidden

Tá, definitivamente não tem como isso ser normal.

Foi aí que eu fui pesquisar sobre o assunto. Achei um post no Reddit falando que isso poderia acontecer quando o aplicativo não estava atualizado. Então foi isso: atualizei e…

ERROR: unable to download video data: HTTP Error 403: Forbidden

:)

Procurei mais a fundo pelo Reddit e descobri que, justamente no dia em que eu fiz a aplicação, o yt-dlp parou de funcionar para downloads no YouTube. Que sorte, ein.

Aparentemente, agora estão corrigindo isso, mas não sei exatamente quanto tempo vai demorar. Então, por enquanto, meu post no LinkedIn vai ter que esperar mais um pouco.

Parando pra pensar… se isso aconteceu exatamente quando eu estava fazendo o meu CLI…

SERÁ QUE FOI CULPA MINHA?!
